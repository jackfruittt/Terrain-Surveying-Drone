\doxysubsection{Detailed Description}
\doxylink{class_terrain_map}{Terrain\+Map} class for incremental 1x1m cell-\/by-\/cell terrain mapping. 

This class handles drone-\/centric grid map operations\+:
\begin{DoxyItemize}
\item Starts with 1x1m grid centred on drone
\item Expands by single 1m cells in flight direction
\item Each cell represents 1m x 1m of terrain 
\end{DoxyItemize}
